**Open Source. Local. Private.**

Bamsense PDF Tools is a comprehensive, client-side PDF manipulation suite designed under the **bamsense.works** brand. Unlike traditional PDF tools that upload your sensitive documents to a cloud server, Bamsense processes everything **100% locally** in your browser.

Your data never leaves your device.

---

## ğŸ›¡ï¸ Privacy Philosophy
- **Zero Server Uploads:** All processing (merging, splitting, compressing) happens using WebAssembly and JavaScript inside your browser memory.
- **Local Only:** No trackers, no data collection, and no cloud storage.
- **Open Source:** Full transparency into how your files are handled.

---

## ğŸš€ Key Features

### 1. Merge PDF
Combine multiple PDFs into one unified document with a drag-and-drop sortable interface.

### 2. Split PDF
Extract specific pages or ranges visually using a high-fidelity "Point & Click" grid selector.

### 3. Organize PDF (The "Light Table")
A professional workspace where you can reorder, rotate, or delete pages with real-time rendered thumbnails.

### 4. Compress PDF
Choose between "Light" metadata stripping or "Extreme" Rasterization (converting text to images) for maximum file size reduction.

### 5. Watermark Studio
A professional design pane to apply text stamps with real-time controls for color, opacity, rotation, and tiled patterns.

### 6. PDF to Image / Image to PDF
Convert entire documents into high-res JPG ZIP archives, or combine your photos into a professional PDF.

### 7. Rotate PDF
A live rotation tool that permanently corrects document orientation with instant visual feedback.

## ğŸƒ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Balagangadhar-Dev/pdf-tools.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
